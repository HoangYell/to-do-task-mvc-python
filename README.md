# TO DO LIST

To do list is an open source built with [Python][0] using the [Basic MVC Architecture][1].

The project has 5 basic functions:

* Create
* Read
* Update
* Delete
* Calendar

## Create

Create a new task

## Read

Read all tasks

## Update

Update a task

## Delete

Delete a task

## Calendar

Show the tasks on a calendar view with: Date, Week, Month

## Technology Stack

* MVC architecture[3]
* Python 3[0]
* JQuery[4]
* Bootstrap 4[5]
* Shelve[6]
* Full calendar[7]
